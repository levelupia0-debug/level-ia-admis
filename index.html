Resend — Configuration email

Le SDK utilise ton compte Resend pour envoyer tous les emails : invitations Watch Party, clés utilisateur, notifications.

Ta clé Resend reste sur Vercel — aucune clé à saisir ici.
Le SDK appelle ton endpoint API Vercel qui envoie les emails de son côté. Colle simplement l'URL complète de ta fonction /api/send ci-dessous. Les templates HTML (Watch Party, Clé, Notification…) sont déjà intégrés dans le handler.
URL complète de ta fonction Vercel qui gère l'envoi Resend. Exemple : https://levelup-ecosystem.com/api/send

Assistant LevelUp IA

Intégrez l'assistant IA dans n'importe quelle app avec une seule ligne de code. Il apparaît comme un widget flottant.

Aperçu en direct
Ouvrir en plein écran

Intégrer le widget dans une app

Le bouton flottant 🤖 apparaît automatiquement sur toutes les pages où le SDK est chargé, dès que tu appelles openAssistant(). Ou intègre un bouton custom :

<!-- 1. Le SDK LevelUp avec une clé qui a le scope "auth" --> <script src="https://levelup-ecosystem.com/sdk.js" data-levelup-key="VOTRE_CLE_SDK"></script> <!-- 2. Ouvrir l'assistant depuis un bouton custom --> <button onclick="LevelUp.openAssistant()"> 🤖 Assistant LevelUp </button> <script> // Ou l'ouvrir automatiquement au chargement LevelUp.openAssistant(); // Fermer le widget LevelUp.closeAssistant(); </script>

Intégrer en iframe fixe dans une page

Si tu veux l'assistant intégré directement dans une zone de ta page (sans widget flottant) :

<!-- Iframe fixe, taille libre --> <iframe src="https://levelupia0-debug.github.io/levelup-assistant/" width="400" height="620" style="border:none;border-radius:16px" allow="microphone; camera" loading="lazy" ></iframe>
Aucun scope spécial requis — l'assistant est disponible sur toutes les clés SDK actives. L'utilisateur n'a pas besoin d'être connecté pour utiliser l'assistant.

Installation

Ajoutez ce script dans le <head> de votre app :

<script src="https://levelup-ecosystem.com/sdk.js" data-levelup-key="VOTRE_CLE_SDK"></script>

Auth — Scope : auth

Connexion / déconnexion / état utilisateur. Inclus sur toutes les clés.

// Vérifier si connecté LevelUp.isLoggedIn(); // → true / false LevelUp.getUser(); // → { uid, name, email, photo, ... } // Ouvrir le popup de connexion (avec Turnstile) LevelUp.openLogin(); // Écouter les événements LevelUp.onLogin((user) => console.log('Connecté :', user.name)); LevelUp.onLogout(() => console.log('Déconnecté')); // Déconnexion + tableau de bord LevelUp.logout(); LevelUp.openDashboard();

Watch Party — Scope : watchparty

Invitations par email pour les Watch Party LevelMovie.

// Envoyer une invitation Watch Party par email await LevelUp.sendWatchPartyInvite({ to : 'ami@email.com', partyCode : 'PARTY-ABC123', movieTitle : 'Inception', inviterName : LevelUp.getUser().name // optionnel }); // → Envoie le type WATCH_PARTY via ton endpoint Vercel (template intégré dans le handler)

Clés utilisateur — Scope : keygen

Générer ou régénérer une clé personnelle pour l'utilisateur connecté. La clé est envoyée par email et n'est JAMAIS affichée en clair dans l'interface.

// Générer une clé pour l'utilisateur (envoyée à son email) const msg = await LevelUp.generateUserKey(); console.log(msg); // → "✅ Clé générée et envoyée par email à user@email.com" // Régénérer la clé (invalide l'ancienne) const msg2 = await LevelUp.regenerateUserKey(); // Voir la clé masquée (6 derniers caractères seulement) LevelUp.getKey(); // → "••••••••••••abc123"

Email custom — Scope : email.send

Envoyer un email HTML personnalisé via ton compte Resend. Tu construis le HTML toi-même.

// Envoyer un email HTML custom via Resend await LevelUp.sendEmail( 'destinataire@email.com', 'Bienvenue sur LevelUp !', // Sujet `<h1>Bonjour ${user.name}</h1> <p>Bienvenue sur la plateforme LevelUp !</p>` // HTML complet ); // Exemple avec une variable utilisateur const user = LevelUp.getUser(); await LevelUp.sendEmail( user.email, '🎉 Ton compte est prêt', `<p>Salut <strong>${user.name}</strong>, bienvenue !</p>` );

Vérifier les scopes

// Vérifier si la clé a un scope avant d'appeler une feature if (LevelUp.hasScope('watchparty')) { await LevelUp.sendWatchPartyInvite({ ... }); } // Voir tous les scopes de la clé active LevelUp.getScopes(); // → ['auth', 'watchparty', 'keygen', ...] // Infos générales sur la clé LevelUp.getSdkInfo(); // → { name, scopes, active }
